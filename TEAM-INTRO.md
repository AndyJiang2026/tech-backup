## 🔄 工作流程

```
用户 → 小马 (chief-agent) → 路由到专业 agent → 汇总回复
```

**我负责**：
1. 接收你的需求
2. 分析任务类型
3. 路由到最合适的下属 agent
4. 汇总结果回复你

**下属 agent 负责**：
- 在各自专业领域深度处理任务
- 保证专业质量和 SLA
