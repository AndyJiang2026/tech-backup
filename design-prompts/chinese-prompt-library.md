## 📋 基础结构

### Prompt 公式

```
[主体描述] + [风格关键词] + [色彩方案] + [构图方式] + [技术参数]
```

### 质量修饰词

| 类别 | 关键词 |
|------|--------|
| **清晰度** | 高清、4K、8K、超清、细腻、精致 |
| **质感** | 电影级、专业摄影、工作室灯光、商业级 |
| **细节** | 细节丰富、精细刻画、层次分明、立体感 |
| **色彩** | 色彩鲜艳、对比度高、饱和度高、色彩饱满 |
| **氛围** | 氛围感强、光影效果、戏剧性光线、柔和光线 |
