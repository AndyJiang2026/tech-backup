## 🎯 使用建议

### 官方发布场景（推荐方案 A）
- 学习强国平台首页 Banner
- 国家电影局官网新闻配图
- 官方新闻发布会背景板
- 传统媒体通稿配图

### 线下活动场景
- **印刷海报**：方案 A（A1/A2尺寸，300DPI）
- **展架/易拉宝**：方案 A 或方案 B
- **VR 体验店装饰**：方案 B

### 线上传播场景
- **微信/微博**：方案 A 或方案 C
- **抖音/快手**：方案 B（更吸引年轻用户）
- **官网首页**：方案 A

## 📐 技术规格

| 参数 | 数值 |
|------|------|
| 画布尺寸 | 800 × 1200 px |
| 宽高比 | 2:3（标准海报比例） |
| 色彩模式 | RGB |
| 文件格式 | PNG（透明背景可选） |
| 屏幕分辨率 | 72 DPI |
| 文件大小 | ~400-600KB/张 |

### 印刷输出建议
如需印刷，请：
1. 联系设计师导出 CMYK 模式文件
2. 分辨率提升至 300 DPI
3. 尺寸调整为 A1 (594×841mm) 或 A2 (420×594mm)
4. 添加 3mm 出血边
5. 选择哑光或丝光相纸（推荐 200g 以上）
