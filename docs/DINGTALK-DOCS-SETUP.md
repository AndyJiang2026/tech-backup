# 钉钉文档权限配置快速指南

## 🎯 目标

让 OpenClaw 能够读写钉钉云文档，实现办公自动化

---

## ⚡ 5 分钟快速检查

### 步骤 1: 登录钉钉开放平台

```
https://open-dev.dingtalk.com/
```

### 步骤 2: 找到你的应用

1. 点击 **"应用开发"** → **"企业内部开发"**
2. 找到应用（AppKey: `dingd1actonjhn5cexyu`）
3. 点击进入应用详情页

### 步骤 3: 检查权限

在应用详情页，找到 **"权限管理"** 标签

**需要的权限**：
- ✅ 基础信息（已有）
- ❓ 文档空间（需要添加）
- ❓ 知识库（需要添加）
- ❓ 云盘（可选）

### 步骤 4: 添加权限（如果缺失）

1. 点击 **"添加权限"**
2. 搜索 **"文档"** 或 **"doc"**
3. 勾选以下权限：
   - `文档空间` - 访问企业文档
   - `知识库` - 访问知识库
4. 点击 **"确定"**

### 步骤 5: 申请权限

1. 添加后状态为 **"待申请"**
2. 点击 **"申请权限"**
3. 填写申请理由（示例）：
   ```
   用于企业办公自动化，需要读取和创建文档，实现会议纪要自动生成、合同审查报告存档等功能
   ```
4. 提交后等待 **企业管理员审批**

### 步骤 6: 发布应用

**重要**：权限变更后必须重新发布！

1. 点击 **"版本管理"**
2. 点击 **"发布"**
3. 确认发布

### 步骤 7: 等待审批

- 企业管理员会在钉钉收到审批通知
- 审批通过后权限生效

## ⚠️ 常见问题

### Q1: 找不到"文档空间"权限选项

**原因**：部分企业账号可能未开通此功能

**解决**：
1. 联系钉钉客服确认企业账号权限
2. 或考虑升级企业版本

### Q2: 权限申请被拒绝

**原因**：企业管理员可能不了解用途

**解决**：
提供更详细的说明：
```
这是企业内部的 AI 助手，用于：
1. 自动整理会议纪要并存档
2. 合同审查报告自动生成
3. 员工快速查询公司制度
4. 周报/月报自动汇总

所有文档操作都在企业内部，不会外泄
```

### Q3: 发布后仍然报错

**错误**：`InvalidAction.NotFound`

**可能原因**：
1. 权限尚未生效（等待 5-10 分钟）
2. API 端点不对
3. 需要 MCP 支持

**解决**：
```bash
# 重启 Gateway
openclaw gateway restart

# 查看日志
openclaw logs --follow | grep dingtalk

# 如果仍然失败，考虑使用 MCP 方案
```
