# IP 授权合同框架要点

## 项目背景
- **授权方**：《长征·英雄》VR 项目运营公司（学习强国为第一出品方）
- **被授权方**：地市级运营商/合作方
- **授权标的**：《长征·英雄》VR 大空间 IP 使用权
- **授权费用**：30 万元/城市 + 门票分账 20 元/张

---

## 一、学习强国出品方权益梳理

### 1.1 核心权益清单

| 权益类型 | 具体内容 | 权利归属 |
|---------|---------|---------|
| **著作权** | VR 内容、剧本、美术设计、音乐音效 | 学习强国/联合出品方 |
| **商标权** | "长征·英雄"品牌标识、LOGO | 学习强国 |
| **专利权** | VR 技术方案、交互设计（如有） | 项目公司 |
| **商业秘密** | 运营数据、客户资源、技术方案 | 项目公司 |
| **衍生开发权** | 续集、改编、衍生品开发 | 学习强国主导 |

### 1.2 授权链条确认
```
学习强国（第一出品方）
    │
    ├──→ 项目运营公司（独家运营授权）
    │       │
    │       └──→ 地市级被授权方（非独家/区域独家）
    │
    └──→ 其他联合出品方（收益分配）
```

### 1.3 权益保护要点
- 确保学习强国对核心 IP 的最终控制权
- 明确项目运营公司的转授权权限
- 建立 IP 使用审批机制
- 设置 IP 滥用违约责任

## 三、门票分账机制法律设计

### 3.1 分账模式选择

| 模式 | 优点 | 缺点 | 推荐度 |
|-----|------|------|-------|
| **固定分账** | 简单透明，易核算 | 无法分享票价上涨收益 | ⭐⭐⭐⭐ |
| **比例分账** | 收益共享，风险共担 | 核算复杂，易产生争议 | ⭐⭐⭐ |
| **保底 + 分成** | 保障最低收益 + 分享增长 | 结构复杂 | ⭐⭐⭐⭐⭐ |

**本项目推荐**：固定分账（20 元/张）+ 最低保底（可选）

### 3.2 分账核算机制

```
分账计算公式：
甲方分账收入 = 实际售票数量 × 20 元/张

实际售票数量 = 总售票数 - 赠票数 - 员工票数 - 退票数

核查机制：
1. 票务系统自动记录
2. 双方每月对账确认
3. 甲方有权委托第三方审计
4. 差异超过 5% 的，审计费用由乙方承担
```

### 3.3 分账支付流程

```
月度分账流程：
每月 1 日 → 系统生成上月报表 → 5 日前乙方确认 → 
10 日前甲方开票 → 15 日前乙方付款 → 甲方确认收款

逾期处理：
- 逾期 1-15 日：按日万分之五滞纳金
- 逾期 15-30 日：书面催款通知
- 逾期 30 日以上：有权暂停技术支持/解除合同
```

### 3.4 防作弊机制

```
技术防范：
1. 统一票务系统，数据实时上传
2. 票务系统与闸机系统联动
3. 异常数据自动预警（如大量退票、赠票）
4. 定期系统升级，堵塞漏洞

合同约束：
1. 明确虚报数据的违约责任（3 倍违约金）
2. 甲方审计权条款
3. 保证金扣除机制
4. 严重违约解除合同
```

## 四、知识产权保护策略

### 4.1 知识产权保护矩阵

| 保护对象 | 保护方式 | 具体措施 | 责任主体 |
|---------|---------|---------|---------|
| **品牌名称** | 商标注册 | 注册第 9/41/42 类商标 | 学习强国 |
| **LOGO 标识** | 商标 + 著作权 | 商标注册 + 美术作品登记 | 学习强国 |
| **VR 内容** | 著作权 | 软件作品 + 视听作品登记 | 项目公司 |
| **技术方案** | 专利/商业秘密 | 专利申请或保密保护 | 项目公司 |
| **运营数据** | 商业秘密 | 保密协议 + 技术措施 | 项目公司 |
| **客户资源** | 商业秘密 | 保密协议 + 竞业限制 | 项目公司 |

### 4.2 商标注册布局

```
核心类别：
- 第 9 类：VR 软件、 downloadable content
- 第 41 类：娱乐服务、VR 体验服务
- 第 42 类：技术服务、软件开发

防御类别：
- 第 16 类：印刷品、宣传材料
- 第 25 类：服装、衍生品
- 第 28 类：游戏器具、玩具
- 第 35 类：广告、商业管理

注册策略：
1. 核心类别全面注册
2. 近似商标防御注册
3. 域名同步注册（.com/.cn/.com.cn）
4. 定期监测，及时异议
```

### 4.3 著作权登记

```
登记内容：
1. VR 体验内容（视听作品）
2. 剧本、故事大纲（文字作品）
3. 角色设计、场景设计（美术作品）
4. 背景音乐、音效（音乐作品）
5. 软件代码（计算机软件）

登记流程：
1. 整理创作材料
2. 填写登记申请表
3. 提交中国版权保护中心
4. 缴纳登记费用
5. 领取登记证书（约 30 个工作日）
```

### 4.4 侵权应对措施

```
监测机制：
1. 定期网络监测（商标、内容侵权）
2. 市场巡查（未经授权的经营行为）
3. 被授权方举报机制
4. 第三方监测服务

维权流程：
发现侵权 → 证据固定 → 律师函警告 → 
行政投诉/诉讼 → 执行赔偿

维权手段：
1. 平台投诉（下架侵权内容）
2. 行政投诉（市场监管局、版权局）
3. 民事诉讼（停止侵权 + 赔偿）
4. 刑事报案（严重侵权）
```

### 4.5 合同中的 IP 保护条款模板

```
知识产权特别条款：

1. 权利保证
甲方保证对授权 IP 拥有完整、合法的权利，有权进行本授权。
如因 IP 权利瑕疵导致乙方损失的，甲方承担赔偿责任。

2. 侵权处理
如发现第三方侵犯授权 IP，甲方负责维权，乙方应配合。
维权所得赔偿，扣除成本后按 [甲方 70% : 乙方 30%] 分配。

3. 乙方侵权
乙方不得擅自使用、注册与授权 IP 相同或近似的标识。
乙方违约的，应立即停止使用，支付违约金 [50 万元]，赔偿甲方损失。

4. 合同终止后
合同终止后 [30] 日内，乙方应销毁所有含有授权 IP 的材料。
乙方继续使用的，按侵权处理，承担法律责任。
```

## 六、合同审核要点清单

### 6.1 授权方审核要点
- [ ] 授权范围是否清晰（地域、期限、内容）
