## 第二条 授权范围

### 2.1 授权内容
甲方授权乙方在授权区域内使用以下授权 IP：
- (a) 使用"长征·英雄"品牌名称、商标及标识
- (b) 使用 VR 体验内容（版本号：[xxx]，详见附件一）
- (c) 使用运营管理系统及技术支持
- (d) 使用统一视觉识别系统（VI）

### 2.2 授权性质
本协议项下授权为 **[□独家 / □非独家]** 授权

### 2.3 授权期限
授权期限为 **[3]** 年，自 **2026 年 X 月 X 日** 起至 **2029 年 X 月 X 日** 止。
期满前 **[90]** 日，双方可协商续约事宜，乙方在同等条件下享有优先续约权。

### 2.4 授权限制
- (a) **地域限制**: 乙方仅限在授权区域内运营，不得跨区域经营
- (b) **用途限制**: 仅限用于 VR 大空间体验项目，不得用于其他商业用途
- (c) **转授权限制**: 未经甲方书面同意，乙方不得将授权转授、分许可给任何第三方
- (d) **修改限制**: 乙方不得擅自修改 VR 内容、品牌形象、技术方案
- (e) **竞争限制**: 授权期限内，乙方不得运营与授权 IP 存在竞争关系的同类项目

## 第三条 费用与支付

### 3.1 固定授权费
- (a) 授权费总额：人民币 **30 万元**（大写：叁拾万元整）
- (b) 支付方式：
  - 首期：合同签订后 **5 个工作日** 内支付 **50%**（15 万元）
  - 二期：项目开业前 **5 个工作日** 内支付 **50%**（15 万元）
- (c) 授权费为 **含税** 价格，甲方开具增值税专用发票

### 3.2 营业收入分账
- (a) 分账标准：按乙方 **营业收入的 [X]%** 或 **每张门票 20 元** 计算（以较高者为准）
- (b) 分账基数：以实际售票数量及营业收入为准（不含赠票、员工票）
- (c) 结算周期：**按月结算**，次月 **10 日** 前双方核对，**15 日** 前乙方支付
- (d) 数据对接：乙方需接入甲方统一的票务管理系统，确保数据真实完整
- (e) 审计权利：甲方有权 **每年一次** 委托第三方审计乙方财务数据，审计费用由甲方承担；如发现乙方少报金额超过 **5%**，审计费用由乙方承担，并按少报金额的 **3 倍** 支付违约金
- (f) 票价管理：
  - 基础票价由甲方统一制定
  - 促销活动需提前 **15 日** 报备甲方
  - 最低票价不得低于 **[XX]** 元

### 3.3 履约保证金
- (a) 保证金金额：人民币 **10 万元**
- (b) 支付时间：合同签订后 **5 个工作日** 内
- (c) 退还条件：合同期满且无违约行为，**30 个工作日** 内无息退还
- (d) 扣除情形：
  - 拖欠分账款项
  - 擅自修改 IP 内容
  - 跨区域经营
  - 其他违约行为导致甲方损失

### 3.4 逾期付款责任 ⭐【新增】
- (a) 乙方逾期付款，每逾期一日，按逾期金额的 **万分之五** 支付滞纳金
- (b) 逾期超过 **30 日**，甲方有权单方解除合同，保证金不予退还
- (c) 逾期超过 **60 日**，甲方有权追索全部应付款项及 **合同总额 20%** 的违约金

## 第四条 双方权利义务

### 4.1 甲方权利义务

**甲方权利**:
- (a) 有权监督乙方的运营质量和服务标准
- (b) 有权检查乙方的票务数据、财务报表和经营情况
- (c) 有权要求乙方整改不符合标准的行为
- (d) 有权在乙方严重违约时单方解除合同
- (e) 有权对授权 IP 进行升级更新，乙方应配合执行

**甲方义务**:
- (a) 提供完整的 IP 授权文件和技术资料
- (b) 提供运营培训和持续技术支持（培训标准见附件二）
- (c) 统一品牌宣传推广，提升品牌影响力
- (d) 及时更新 VR 内容版本，保持内容竞争力
- (e) 保证授权 IP 不侵犯第三方合法权益

### 4.2 乙方权利义务

**乙方权利**:
- (a) 在授权范围内按约定使用授权 IP
- (b) 获得甲方的培训和运营支持
- (c) 参与甲方组织的联合营销活动
- (d) 优先获得续授权利
- (e) 有权要求甲方按约定提供技术支持

**乙方义务**:
- (a) 按约定支付授权费和分账款项
- (b) 维护授权 IP 的品牌形象和商业声誉
- (c) 按甲方标准提供运营服务，确保用户体验
- (d) 定期向甲方报送经营数据和财务报表
- (e) 配合甲方的监督检查和审计工作
- (f) 未经授权方书面同意，不得将授权 IP 用于质押、担保等处置行为

## 第五条 知识产权 ⭐【强化】

### 5.1 权利归属
- (a) 授权 IP 的全部知识产权归甲方或甲方指定方所有
- (b) 乙方仅在授权期限内获得使用权，不享有任何所有权
- (c) 乙方在授权范围内使用授权 IP 产生的任何成果（含改进、衍生作品），知识产权归 **甲方所有**

### 5.2 知识产权保护
- (a) 乙方发现第三方侵犯授权 IP 时，应立即通知甲方，并配合甲方维权
- (b) 乙方不得对授权 IP 进行反向工程、反编译、反汇编
- (c) 乙方不得申请与授权 IP 相同或近似的商标、专利、著作权
- (d) 乙方不得注册与授权 IP 相同或近似的域名、社交媒体账号

### 5.3 侵权责任
- (a) 甲方保证授权 IP 不侵犯第三方合法权益
- (b) 如因授权 IP 侵权导致乙方被第三方索赔，甲方负责处理并承担赔偿
- (c) 赔偿上限：**授权费总额的 2 倍** 或 **100 万元**（以较低者为准）
- (d) 如授权 IP 被认定侵权，甲方应替换为不侵权的同等 IP，或退还授权费并赔偿损失

## 第六条 保密条款 ⭐【延长】

### 6.1 保密信息范围
- (a) 技术资料：VR 内容、源代码、算法、技术方案
- (b) 商业信息：商业计划、客户信息、财务数据、运营策略
- (c) 协议内容：本协议内容及履行过程中知悉的其他信息
- (d) 其他信息：甲方明确标注为"保密"的信息

### 6.2 保密义务
- (a) 仅限授权人员接触保密信息
- (b) 采取合理保密措施（不低于保护自身保密信息的标准）
- (c) 不得向任何第三方披露保密信息
- (d) 合同终止后 **返还或销毁** 全部保密信息载体

### 6.3 保密期限 ⭐【修改】
保密期限：自合同签订之日起 **[5] 年**，或至保密信息公开之日止（以较长者为准）

### 6.4 例外情形
以下情形不构成保密义务违反：
- (a) 信息已公开或成为公共知识
- (b) 依法应向司法机关、监管机构披露
- (c) 经甲方书面同意披露

## 第七条 违约责任 ⭐【量化】

### 7.1 一般违约
任何一方违反本协议约定，应向守约方支付违约金 **人民币 20 万元**

### 7.2 特别违约
以下情形视为严重违约，违约金为 **合同总额的 30%**：
- (a) 乙方逾期付款超过 30 日
- (b) 乙方擅自转授权、分许可
- (c) 乙方擅自修改 IP 内容或品牌形象
- (d) 乙方跨区域经营
- (e) 乙方运营竞争项目
- (f) 乙方泄露核心商业秘密

### 7.3 损失赔偿
- (a) 违约金不足以弥补守约方损失的，违约方应赔偿差额
- (b) 损失包括但不限于：直接损失、预期利益损失、维权费用（律师费、诉讼费、保全费、差旅费等）

### 7.4 继续履行
违约方支付违约金后，守约方仍有权要求违约方继续履行合同义务

## 第八条 合同解除 ⭐【完善】

### 8.1 协商解除
双方协商一致，可以书面形式解除本协议

### 8.2 单方解除权
有下列情形之一的，守约方有权单方解除合同：
- (a) 一方严重违约，经催告后 **15 日** 内未纠正
- (b) 乙方逾期付款超过 **30 日**
- (c) 乙方擅自转授权、修改 IP、跨区域经营
- (d) 乙方破产、解散、被吊销营业执照
- (e) 因不可抗力导致合同目的无法实现超过 **90 日**

### 8.3 解除程序
- (a) 解除方应书面通知对方，通知到达时合同解除
- (b) 对方有异议的，可在 **15 日** 内向法院起诉确认解除效力

### 8.4 解除后果
合同解除后：
- (a) 乙方立即停止使用授权 IP
- (b) 乙方 **15 日** 内返还或销毁全部授权资料
- (c) 保密义务继续有效
- (d) 已支付费用 **不予退还**（因甲方违约解除的除外）
- (e) 违约方按本协议第七条承担违约责任
