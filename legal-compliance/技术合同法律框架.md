## 二、技术授权合同模板

### 2.1 合同要点

| 条款 | 核心内容 | 注意事项 |
|------|----------|----------|
| 授权标的 | 专利/软著/技术秘密 | 明确清单、附件列明 |
| 授权范围 | 独占/排他/普通许可 | 根据商业策略选择 |
| 授权地域 | 中国大陆/全球 | 与商标布局匹配 |
| 授权期限 | 固定年限/永久 | 考虑技术生命周期 |
| 授权费用 | 一次性/分期/提成 | 明确计算方式 |
| 技术支持 | 培训内容、响应时间 | SLA 标准 |
| 侵权责任 | 侵权处理、赔偿限额 | 风险分配 |
| 合同终止 | 终止条件、后续处理 | 过渡期安排 |

### 2.2 核心条款模板

#### 第一条 授权标的

```
1.1 甲方授权乙方使用的技术标的如下：
    (1) 发明专利：[专利名称]，专利号：[ZL XXXX.X]
    (2) 软件著作权：[软件名称]，登记号：[2026SRXXXXXX]
    (3) 技术秘密：[技术名称]，详见保密附件

1.2 甲方保证对上述技术标的享有完整、合法的处分权，
    不存在权利瑕疵或第三方权利主张。
```

#### 第二条 授权范围

```
2.1 授权类型：[独占许可/排他许可/普通许可]
    - 独占许可：仅乙方可使用，甲方不得使用
    - 排他许可：甲乙双方可使用，甲方不得再授权第三方
    - 普通许可：甲方可授权多方使用

2.2 授权地域：[中国大陆/全球/特定区域]

2.3 授权领域：[XR 应用/AIGC 应用/全领域]

2.4 授权期限：自 [2026] 年 [X] 月 [X] 日起至 [2029] 年 [X] 月 [X] 日止，
    共计 [3] 年。期满前 [3] 个月可协商续约。
```

#### 第三条 授权费用

```
3.1 授权费用总额：人民币 [XXX] 万元（大写：[XXX]）

3.2 支付方式：
    □ 一次性支付：合同签订后 [X] 个工作日内支付
    □ 分期支付：
        - 首期：合同签订后 [X] 个工作日内支付 [X]%
        - 二期：技术交付后 [X] 个工作日内支付 [X]%
        - 尾期：验收合格后 [X] 个工作日内支付 [X]%
    □ 提成支付：按乙方销售收入的 [X]% 提成，按 [季度/年度] 结算

3.3 甲方账户信息：
    户名：[公司名称]
    开户行：[银行名称]
    账号：[银行账号]
```

#### 第四条 技术交付

```
4.1 交付内容：
    (1) 技术文档：设计文档、API 文档、使用手册
    (2) 源代码：[是/否]，语言：[XXX]，版本：[XXX]
    (3) 培训服务：[X] 人天，地点：[XXX]

4.2 交付时间：合同签订后 [X] 个工作日内

4.3 交付方式：[加密 U 盘/安全传输/现场交付]

4.4 验收标准：
    (1) 功能测试通过率≥[95]%
    (2) 性能指标达到附件约定标准
    (3) 文档完整、准确
```

#### 第五条 知识产权

```
5.1 授权技术的所有权归甲方所有，乙方仅获得使用权。

5.2 乙方在授权范围内使用授权技术产生的改进成果：
    □ 归甲方所有
    □ 归乙方所有
    □ 双方共有（甲方 [X]%，乙方 [X]%）

5.3 乙方不得对授权技术进行反向工程、反编译或反汇编。

5.4 乙方不得将授权技术用于本合同约定范围之外的用途。
```

#### 第六条 保密条款

```
6.1 保密信息范围：
    (1) 技术资料、源代码、算法
    (2) 商业计划、客户信息、财务数据
    (3) 本合同内容及履行过程中知悉的其他信息

6.2 保密期限：自合同签订之日起 [5] 年，或至信息公开之日止。

6.3 保密义务：
    (1) 仅限授权人员接触
    (2) 采取合理保密措施
    (3) 不得向第三方披露
    (4) 合同终止后返还或销毁
```

#### 第七条 侵权责任

```
7.1 甲方保证授权技术不侵犯第三方知识产权。

7.2 如因授权技术侵权导致乙方被第三方索赔：
    (1) 甲方负责处理侵权纠纷
    (2) 甲方承担赔偿金、诉讼费、律师费
    (3) 赔偿上限：[授权费用总额的 X 倍/XXX 万元]

7.3 如授权技术被认定侵权：
    (1) 甲方应替换为不侵权的同等技术
    (2) 或退还授权费用并赔偿损失
    (3) 乙方有权解除合同
```

#### 第八条 违约责任

```
8.1 任何一方违约，应向守约方支付违约金 [XXX] 万元。

8.2 违约金不足以弥补损失的，违约方应赔偿差额。

8.3 乙方逾期付款，按日支付逾期金额 [万分之五] 的滞纳金。

8.4 甲方逾期交付，按日支付授权费用 [万分之五] 的违约金。
```

#### 第九条 合同终止

```
9.1 合同终止情形：
    (1) 授权期限届满未续约
    (2) 双方协商一致解除
    (3) 一方严重违约，守约方解除
    (4) 不可抗力导致合同无法履行

9.2 合同终止后：
    (1) 乙方停止使用授权技术
    (2) 乙方返还或销毁技术资料
    (3) 保密义务继续有效
    (4) 已支付费用 [不退还/按比例退还]
```

### 2.3 附件清单

| 附件编号 | 附件名称 | 说明 |
|----------|----------|------|
| 附件一 | 授权技术清单 | 专利号、软著号、技术描述 |
| 附件二 | 技术标准与规范 | 性能指标、测试标准 |
| 附件三 | 交付物清单 | 文档、代码、培训材料 |
| 附件四 | 保密信息范围 | 具体保密内容 |
| 附件五 | 技术服务 SLA | 响应时间、服务标准 |

## 三、SaaS 订阅服务协议模板

### 3.1 合同要点

| 条款 | 核心内容 | 注意事项 |
|------|----------|----------|
| 服务内容 | 平台功能、服务等级 | 明确 SLA 标准 |
| 订阅费用 | 计费方式、支付周期 | 年付/月付、阶梯定价 |
| 账号管理 | 用户数、权限管理 | 并发数、账号转让 |
| 数据安全 | 数据归属、备份恢复 | 数据可携带性 |
| 服务中断 | 免责情形、补偿标准 | SLA 赔偿 |
| 合同变更 | 功能变更、价格调整 | 提前通知期 |
| 合同终止 | 退费政策、数据导出 | 过渡期安排 |

### 3.2 核心条款模板

#### 第一条 服务内容

```
1.1 甲方向乙方提供以下 SaaS 服务：
    (1) XR 内容创作平台：[功能描述]
    (2) AIGC 生成工具：[功能描述]
    (3) 数据分析服务：[功能描述]
    详见《服务功能说明书》（附件一）

1.2 服务方式：通过互联网提供云端服务
    服务地址：https://[域名].com

1.3 服务时间：7×24 小时（计划维护除外）
```

#### 第二条 服务等级协议 (SLA)

```
2.1 服务可用性：
    - 月度可用性≥99.5%
    - 计算公式：(月度总分钟数 - 不可用分钟数) / 月度总分钟数

2.2 性能指标：
    - 页面响应时间：<3 秒
    - API 响应时间：<500 毫秒
    - 并发用户数：≥[XXX] 人

2.3 服务补偿：
    | 可用性 | 补偿标准 |
    |--------|----------|
    | 99%-99.5% | 当月服务费 10% |
    | 95%-99% | 当月服务费 25% |
    | <95% | 当月服务费 50% |

2.4 免责情形：
    (1) 乙方网络或设备故障
    (2) 乙方操作不当
    (3) 不可抗力
    (4) 计划内维护（提前 48 小时通知）
```

#### 第三条 订阅费用

```
3.1 计费模式：
    □ 按用户数：[XXX] 元/用户/月，最低 [X] 用户
    □ 按功能模块：基础版 [XXX] 元/月，专业版 [XXX] 元/月
    □ 按使用量：[XXX] 元/[调用次数/存储空间/时长]
    □ 混合计费：[具体说明]

3.2 支付周期：
    □ 月付：每月 [X] 日前支付
    □ 季付：每季度首月 [X] 日前支付
    □ 年付：每年 [X] 月 [X] 日前支付（享 [X] 折优惠）

3.3 价格调整：
    - 甲方调整价格需提前 [30] 天书面通知
    - 已支付周期内价格不变
    - 乙方不同意可解除合同，按比例退费
```

#### 第四条 账号与权限

```
4.1 乙方账号信息：
    - 管理员账号：[X] 个
    - 普通用户账号：[X] 个
    - 可增减，按实际使用结算

4.2 权限管理：
    - 管理员：全权限（用户管理、配置、数据导出）
    - 普通用户：使用权限
    - 访客：只读权限

4.3 账号安全：
    - 乙方负责账号密码保管
    - 不得转借、出租、出售账号
    - 发现泄露立即通知甲方
```

#### 第五条 数据权利

```
5.1 数据归属：
    - 乙方上传、生成的数据归乙方所有
    - 甲方仅在服务范围内处理数据
    - 合同终止后乙方可导出全部数据

5.2 数据备份：
    - 甲方每日自动备份
    - 保留期限：[30] 天
    - 乙方可申请恢复历史数据

5.3 数据保密：
    - 甲方不得查看、使用乙方数据（除非必要）
    - 不得向第三方披露
    - 法律法规要求除外
```

#### 第六条 知识产权

```
6.1 平台知识产权归甲方所有。

6.2 乙方使用平台生成的内容：
    □ 归乙方所有（推荐）
    □ 双方共有
    □ 归甲方所有，乙方获使用权

6.3 乙方保证上传内容不侵犯第三方权利。

6.4 甲方有权对违规内容进行删除、屏蔽。
```

#### 第七条 合同期限与终止

```
7.1 合同期限：自 [2026] 年 [X] 月 [X] 日起至 [2027] 年 [X] 月 [X] 日止。

7.2 自动续约：期满前 [30] 天未书面通知终止，自动续约 [1] 年。

7.3 乙方解除权：
    - 提前 [30] 天书面通知
    - 已支付费用按剩余天数退还

7.4 甲方解除权：
    - 乙方逾期付款超过 [30] 天
    - 乙方严重违反使用规则
    - 提前 [30] 天通知，按比例退费

7.5 合同终止后：
    - 服务立即停止
    - 乙方可在 [30] 天内导出数据
    - 逾期数据将被删除
```

## 四、定制开发合同模板

### 4.1 合同要点

| 条款 | 核心内容 | 注意事项 |
|------|----------|----------|
| 需求范围 | 功能清单、技术规格 | 需求变更流程 |
| 开发周期 | 里程碑、交付时间 | 延期责任 |
| 开发费用 | 总价、支付节点 | 与里程碑挂钩 |
| 验收标准 | 测试用例、验收流程 | 验收不通过处理 |
| 知识产权 | 成果归属、源码交付 | 核心条款 |
| 维护服务 | 保修期、响应时间 | 免费/收费 |
| 需求变更 | 变更流程、费用调整 | 避免范围蔓延 |

### 4.2 核心条款模板

#### 第一条 开发内容

```
1.1 项目名称：[XR/AIGC 定制开发项目]

1.2 开发内容：
    (1) 功能模块：[详见需求规格说明书]
    (2) 技术架构：[技术栈说明]
    (3) 交付物：源代码、文档、部署包

1.3 技术规格：
    - 开发语言：[XXX]
    - 数据库：[XXX]
    - 部署环境：[XXX]
    - 兼容性：[浏览器/设备要求]
```

#### 第二条 开发周期

```
2.1 项目周期：[X] 个工作日，自 [2026] 年 [X] 月 [X] 日至 [2026] 年 [X] 月 [X] 日

2.2 里程碑计划：
    | 阶段 | 交付物 | 时间 | 付款比例 |
    |------|--------|------|----------|
    | 需求确认 | 需求规格说明书 | 第 [X] 周 | [20]% |
    | 设计完成 | 系统设计文档 | 第 [X] 周 | [20]% |
    | 开发完成 | 测试版本 | 第 [X] 周 | [30]% |
    | 验收通过 | 正式版本 | 第 [X] 周 | [25]% |
    | 质保期满 | 质保报告 | 第 [X] 月 | [5]% |

2.3 延期处理：
    - 甲方原因延期：工期顺延
    - 乙方原因延期：按日支付合同总额 [万分之五] 违约金
    - 延期超过 [30] 天，甲方有权解除合同
```

#### 第三条 开发费用

```
3.1 合同总额：人民币 [XXX] 万元（大写：[XXX]）

3.2 费用构成：
    - 人工成本：[XXX] 万元（[X] 人天 × [XXX] 元/人天）
    - 技术成本：[XXX] 万元（第三方组件、云服务）
    - 管理成本：[XXX] 万元（项目管理、质量保证）
    - 税费：[XXX] 万元（[X]% 增值税）

3.3 支付方式：按里程碑支付（见 2.2 条）

3.4 需求变更费用：
    - 变更工作量<[X] 人天：免费
    - 变更工作量≥[X] 人天：按 [XXX] 元/人天计费
    - 变更需双方书面确认
```

#### 第四条 验收标准

```
4.1 验收依据：
    (1) 本合同及附件
    (2) 需求规格说明书
    (3) 系统设计文档
    (4) 行业相关标准

4.2 验收流程：
    (1) 乙方提交验收申请及测试报告
    (2) 甲方在 [X] 个工作日内组织验收
    (3) 验收通过：签署验收报告
    (4) 验收不通过：乙方在 [X] 个工作日内整改

4.3 验收标准：
    - 功能测试通过率≥[95]%
    - 严重缺陷数=0
    - 一般缺陷数≤[X] 个
    - 性能指标达到约定标准

4.4 视为验收：
    - 甲方在 [X] 个工作日内未组织验收
    - 或已实际使用系统
    - 视为验收通过
```

#### 第五条 知识产权

```
5.1 知识产权归属：
    □ 归甲方所有（甲方付费购买）
    □ 归乙方所有，甲方获永久使用权
    □ 双方共有（甲方 [X]%，乙方 [X]%）
    □ 背景知识产权归原所有方，新增知识产权 [协商]

5.2 源码交付：
    - 交付内容：完整源代码、构建脚本、依赖清单
    - 交付时间：验收后 [X] 个工作日内
    - 交付方式：[加密 U 盘/Git 仓库]

5.3 乙方保证：
    - 开发成果不侵犯第三方知识产权
    - 使用的开源组件符合许可协议
    - 如侵权，乙方负责处理并赔偿
```

#### 第六条 维护服务

```
6.1 免费质保期：验收合格后 [12] 个月

6.2 质保范围：
    - 系统缺陷修复
    - 性能优化
    - 安全漏洞修复
    - 不包括：新功能开发、需求变更

6.3 服务响应：
    | 问题级别 | 响应时间 | 解决时间 |
    |----------|----------|----------|
    | 紧急（系统瘫痪） | 2 小时 | 24 小时 |
    | 重要（功能不可用） | 4 小时 | 48 小时 |
    | 一般（轻微问题） | 24 小时 | 5 工作日 |

6.4 质保期后服务：
    - 可签订维护合同
    - 收费标准：[XXX] 元/年 或 [XXX] 元/人天
```

### 4.3 附件清单

| 附件编号 | 附件名称 | 说明 |

## 五、技术保密协议 (NDA) 模板

### 5.1 合同要点

| 条款 | 核心内容 | 注意事项 |
|------|----------|----------|
| 保密信息 | 定义范围、载体形式 | 尽量具体 |
| 保密义务 | 保密措施、使用限制 | 可执行性 |
| 保密期限 | 起止时间、例外情形 | 合理期限 |
| 违约责任 | 违约金、赔偿范围 | 威慑力 |
| 返还销毁 | 返还方式、时间节点 | 可验证 |

### 5.2 核心条款模板

#### 第一条 保密信息定义

```
1.1 保密信息包括：
    (1) 技术信息：源代码、算法、设计文档、技术方案
    (2) 商业信息：商业计划、客户名单、财务数据、定价策略
    (3) 运营信息：管理制度、流程规范、培训材料
    (4) 其他信息：标注"保密"或应知为保密的信息

1.2 保密信息载体：
    - 书面文件、电子文档
    - 口头披露（需事后书面确认）
    - 演示、展示中披露的信息
```

#### 第二条 保密义务

```
2.1 保密义务：
    (1) 严格保密，不得向第三方披露
    (2) 仅限授权人员接触
    (3) 采取合理保密措施（不低于自身保密信息）
    (4) 不得用于本合同目的之外的用途

2.2 授权人员：
    - 直接参与项目的人员
    - 需知悉保密信息的人员
    - 已签署保密协议的人员

2.3 强制披露：
    - 法律法规要求披露
    - 政府部门要求披露
    - 应提前通知披露方
    - 仅限必要范围
```

#### 第三条 保密期限

```
3.1 保密期限：自本协议签署之日起 [5] 年

3.2 核心商业秘密（源代码、核心算法）：永久保密

3.3 保密期限届满后：
    - 保密义务终止
    - 但不得恶意使用损害披露方利益
```

#### 第四条 例外情形

```
4.1 以下信息不属于保密信息：
    (1) 披露时已为公众所知
    (2) 非因接收方过错后成为公众所知
    (3) 接收方在披露前已合法知悉
    (4) 接收方独立开发获得
    (5) 从第三方合法获得（无保密义务）
```

#### 第五条 违约责任

```
5.1 违约方应支付违约金人民币 [XXX] 万元

5.2 违约金不足以弥补损失的，应赔偿差额

5.3 损失包括：
    - 直接经济损失
    - 可得利益损失
    - 维权费用（律师费、诉讼费、调查费）

5.4 违约方应立即停止违约行为

5.5 守约方有权解除合同
```

#### 第六条 返还与销毁

```
6.1 合同终止或披露方要求时，接收方应：
    (1) 返还全部保密信息载体
    (2) 删除电子存储的保密信息
    (3) 出具书面销毁证明

6.2 返还/销毁期限：[X] 个工作日内

6.3 法律法规要求保留的，可保留但继续保密
```

## 七、争议解决条款

### 7.1 推荐条款

```
凡因本合同引起的或与本合同有关的任何争议，双方应友好协商解决；
协商不成的，按以下第 [X] 种方式处理：

(1) 提交 [XXX] 仲裁委员会，按照申请仲裁时该会现行有效的仲裁规则
    进行仲裁。仲裁裁决是终局的，对双方均有约束力。

(2) 向 [甲方/乙方] 所在地有管辖权的人民法院提起诉讼。

(3) 向合同签订地有管辖权的人民法院提起诉讼。
```

### 7.2 选择建议

| 方式 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| 仲裁 | 保密、专业、一裁终局 | 费用较高、无上诉 | 技术合同、商业秘密 |
| 诉讼 | 费用较低、可上诉 | 公开审理、周期长 | 一般合同 |
| 甲方所在地 | 便利甲方 | 乙方可能抵触 | 甲方强势 |
| 乙方所在地 | 便利乙方 | 甲方可能抵触 | 乙方强势 |
| 合同签订地 | 中立 | 可能双方都不便 | 平等协商 |
