# 🎬 视频编辑方案

**编辑需求**: 适合手机竖屏观看，配上马蹄声效和欢快背景音乐，分镜头有电影感

---

## 📋 编辑要求

### 1. 画面调整
- **输出格式**: 竖屏 9:16 (1080x1920)
- **适配方式**: 原视频居中，两侧添加模糊背景或渐变
- **帧率**: 保持原视频帧率或调整为 30fps

### 2. 音频配置
- **马蹄声效**: 在视频开头或转场时添加
- **背景音乐**: 欢快、轻松的曲风
- **音量平衡**: 背景音乐不盖过原视频音频（如果有）

### 3. 电影感分镜头
- **转场效果**: 淡入淡出、交叉溶解
- **镜头节奏**: 根据内容节奏调整
- **色彩调整**: 增加对比度，适当调色

## 🛠️ 推荐工具

### 方案 A: 使用剪映/CapCut（推荐）⭐

**优点**:
- 免费、易用
- 内置丰富的音效和音乐库
- 自动适配竖屏格式
- 电影感滤镜和转场效果

**操作步骤**:
1. 导入原视频
2. 选择"比例" → 9:16
3. 添加"马蹄声"音效（音效库搜索）
4. 添加欢快背景音乐（音乐库搜索"欢快"）
5. 添加转场效果（淡入淡出）
6. 应用电影感滤镜
7. 导出视频

---

### 方案 B: 使用 FFmpeg（命令行）

**安装**:
```bash
# 如果系统支持
sudo yum install ffmpeg
# 或
brew install ffmpeg
```

**处理脚本**:
```bash
# 1. 转换为竖屏（添加模糊背景）
ffmpeg -i input.mp4 -vf "
  scale=1080:1920:force_original_aspect_ratio=decrease,
  pad=1080:1920:(ow-iw)/2:(oh-ih)/2,
  format=yuv420p
" -c:a copy output_vertical.mp4

# 2. 添加音频（需要音频文件）
ffmpeg -i output_vertical.mp4 -i horse_hooves.mp3 -i happy_bgm.mp3 -filter_complex "
  [1:a][2:a]amix=inputs=2:duration=first:dropout_transition=2[a]
" -map 0:v -map "[a]" -c:v libx264 -c:a aac output_final.mp4
```
