## 🚀 快速开始

### 方式一：浏览器演示（推荐）

1. **打开文件**
   ```bash
   # Linux/Mac
   xdg-open "长征·英雄 2026-2029 三年发展规划.html"
   
   # 或直接用浏览器打开
   # Chrome / Edge / Safari / Firefox
   ```

2. **键盘控制**
   - `→` / `空格` / `PageDown`：下一页
   - `←` / `PageUp`：上一页
   - `Home`：第一页
   - `End`：最后一页

3. **全屏演示**
   - 点击底部"全屏演示"按钮
   - 或按 `F11` 键

### 方式二：导出 PDF

1. 在浏览器中打开 HTML 文件
2. 点击底部"打印/导出 PDF"按钮
3. 选择"另存为 PDF"
4. 页面设置：A4 横向

### 方式三：转换为 PowerPoint

由于技术限制，当前交付的是 HTML 格式。如需.pptx 格式：

1. **方法 A**：HTML → PDF → PPT
   - 先导出 PDF（见方式二）
   - 使用 Adobe Acrobat 或在线工具转换

2. **方法 B**：按设计稿重新制作
   - 参考"设计说明文档.md"中的设计规范
   - 在 PowerPoint/Keynote 中按设计稿制作
