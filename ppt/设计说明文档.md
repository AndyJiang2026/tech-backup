## 📊 图表规范

### 1. 柱状图
- **第一曲线**：蓝色系 `#0066ff` → `#003366`
- **第二曲线**：橙色系 `#ff6600` → `#663300`
- **圆角**：4px（顶部）
- **宽度**：50px
- **间距**：8px

### 2. 折线图
- **颜色**：红色或橙色 `#ff6600`
- **线宽**：3px
- **数据点**：白色填充，描边 2px
- **用途**：突出增长趋势

### 3. 饼图
- **多色区分**：使用不同色相
- **标注**：百分比 + 数值
- **图例**：右侧垂直排列

### 4. 时间轴
- **布局**：横向
- **线条**：渐变色（蓝→橙）
- **节点**：白色圆点，边框区分曲线
- **箭头**：指引方向

## 📁 文件说明

### 交付文件
1. **长征·英雄 2026-2029 三年发展规划.html**
   - 交互式 HTML PPT
   - 40 页完整内容
   - 支持键盘控制（← → Home End）
   - 支持全屏演示
   - 支持打印/导出 PDF

2. **设计说明文档.md**（本文件）
   - 设计规范说明
   - 使用说明

3. **generate_ppt.py**
   - PPT 生成脚本
   - 可用于重新生成或修改

### 使用方法

#### 浏览器演示
1. 用 Chrome/Edge/Safari 打开 HTML 文件
2. 使用按钮或键盘控制翻页：
   - **→ / 空格 / PageDown**：下一页
   - **← / PageUp**：上一页
   - **Home**：第一页
   - **End**：最后一页
3. 点击"全屏演示"进入全屏模式

#### 导出 PDF
1. 在浏览器中打开 HTML 文件
2. 点击"打印/导出 PDF"按钮
3. 选择"另存为 PDF"
4. 调整页面设置为 A4 横向

#### 转换为 PowerPoint
如需转换为.pptx 格式，建议：
1. 使用浏览器打印功能导出 PDF
2. 使用 Adobe Acrobat 或在线工具将 PDF 转为 PPT
3. 或使用专业 PPT 制作工具按设计稿重新制作
