## 🎯 新增配置

### 1. qwen-plus 经济模型 ✅

**特点**:
- 上下文：128K tokens
- 成本：¥0.002/千 tokens（输入）
- 速度：快
- 适用：简单问题/快速响应

**使用场景**:
```
用户："今天天气如何？"
→ 自动选择 qwen-plus
→ 快速响应
→ 成本最低（比 qwen3.5-plus 便宜 50%）
```

### 2. 故障转移配置 ✅

**配置**:
```json
{
  "failover": {
    "enabled": true,
    "timeout": 30000,
    "retries": 3,
    "fallback_order": ["dashscope-coding", "dashscope", "minimax"],
    "rules": [
      {"condition": "timeout", "action": "switch_to_next_provider"},
      {"condition": "rate_limit", "action": "retry_with_backoff"},
      {"condition": "error_5xx", "action": "switch_to_next_provider"}
    ]
  }
}
```

**工作流程**:
```
dashscope-coding 超时
  ↓
自动切换到 dashscope
  ↓
继续响应（用户无感知）
```
