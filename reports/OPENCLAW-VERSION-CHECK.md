## 🎯 更新建议

### 升级优先级：**中**

**理由**：
- ✅ 当前版本稳定运行
- ⚠️ 最新版本可能包含 bug 修复和新功能
- ⚠️ 建议评估更新日志后决定

### 升级步骤

**1. 查看更新日志**：
```bash
npm view openclaw --json | jq '.changelog'
```

**2. 备份当前配置**：
```bash
cp -r /home/admin/.openclaw /home/admin/.openclaw.backup.$(date +%Y%m%d)
```

**3. 执行升级**：
```bash
# 全局升级
npm install -g openclaw

# 或指定版本
npm install -g openclaw@2026.4.9
```

**4. 验证升级**：
```bash
openclaw --version
openclaw gateway restart
```
