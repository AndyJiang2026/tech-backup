- ✅ 选择合适升级窗口

**升级步骤**：

#### 1. 升级前准备（30 分钟）

```bash
# 1.1 完整备份
cp -r /home/admin/.openclaw /home/admin/.openclaw.backup.$(date +%Y%m%d_%H%M%S)
echo "✅ 配置已备份"

# 1.2 导出当前配置
openclaw config export > openclaw-config-backup.json
echo "✅ 配置已导出"

# 1.3 记录当前状态
openclaw --version > version-before.txt
openclaw gateway status > gateway-status-before.txt
echo "✅ 状态已记录"
```

#### 2. 查看更新日志（15 分钟）

```bash
# 2.1 查看 NPM 包信息
npm view openclaw@2026.4.9 --json | jq '.changelog'

# 2.2 访问 GitHub 查看 Release
# https://github.com/openclaw/openclaw/releases/tag/v2026.4.9

# 2.3 查看破坏性变更
npm view openclaw@2026.4.9 --json | jq '.breakingChanges'
```

#### 3. 执行升级（10 分钟）

```bash
# 3.1 停止 Gateway
openclaw gateway stop

# 3.2 升级 NPM 包
npm install -g openclaw@2026.4.9

# 3.3 验证安装
openclaw --version

# 3.4 启动 Gateway
openclaw gateway start
```

#### 4. 验证功能（30 分钟）

```bash
# 4.1 检查 Gateway 状态
openclaw gateway status

# 4.2 测试核心功能
openclaw status

# 4.3 验证配置完整性
cat /home/admin/.openclaw/openclaw.json

# 4.4 测试 Agent 功能
# 发送测试消息验证
```

#### 5. 回滚方案（如失败）

```bash
# 5.1 停止 Gateway
openclaw gateway stop

# 5.2 恢复旧版本
npm install -g openclaw@2026.3.13

# 5.3 恢复配置
cp -r /home/admin/.openclaw.backup.* /home/admin/.openclaw

# 5.4 重启 Gateway
openclaw gateway start

# 5.5 验证恢复
openclaw --version
```
