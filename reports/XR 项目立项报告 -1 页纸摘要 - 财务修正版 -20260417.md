# XR 项目立项报告 - 1 页纸汇报摘要
# （财务修正版·最终）

**项目名称**：联合学习强国《长征·英雄》VR 大空间项目  
**呈报日期**：2026 年 4 月 17 日  
**呈报单位**：内蒙古广播电视台融媒体中心

---

## 🎯 核心主张

**联合学习强国，盘活旧台闲置资产，打造全区首个"文旅 + 科技 + 思政"XR 沉浸式数智创研中心**

以长征胜利 90 周年（2026）和内蒙古自治区成立 80 周年（2027）为双节点驱动，复制河南台《唐宫夜宴》成功模式，实现社会效益与经济效益双丰收。

---

## 📊 关键信息速览

| 维度 | 核心信息 |
|-----|---------|
| **项目定位** | 存量资产盘活 + 数字化转型示范工程 + 学习强国联合项目 |
| **对标案例** | 河南台《唐宫夜宴》VR 项目（投入 300 万，年营收 2000 万+，ROI 1:6.7） |
| **战略窗口** | 长征 90 周年（2026）+ 内蒙 80 周年（2027）双节点驱动 |
| **合作模式** | 学习强国城市合伙人协议（甲：学习强国 / 乙：未来新视界 / 丙：内蒙古台） |
| **授权区域** | 呼和浩特市（首期），后续可扩展至包头、鄂尔多斯等全区 |
| **投资成本** | IP 授权费 50 万（一次性）+ 硬件 18 万 + 改造 100 万 = **168 万元**（仅 VR 体验区） |
| **分账方式** | 30 元/台（散客）+ 15 元/人（学生研学），按月结算，无保底压力 |
| **合作期限** | 3 年（可续约） |
| **财务测算** | 年营收 900-1200 万，年利润 330-630 万，**回本周期 4-7 个月** ⭐⭐⭐ |
| **ROI（3 年）** | **1:8.8**（**远超河南台 1:6.7**）⭐⭐⭐ |
| **政策依据** | 台内 2026 年度实施方案第 18、19、33 条直接支撑 |
