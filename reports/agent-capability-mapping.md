## ⚠️ 注意事项

### 技能使用原则
1. **按需激活** - 简单问题不启用复杂分析
2. **框架优先** - 先选框架再分析
3. **多角度验证** - 避免单一视角
4. **数据支撑** - 结论必须有证据

### Agent 协作原则
1. **Leader 统筹** - 强国小马负责任务分发
2. **专业优先** - 专业问题优先对应 Agent
3. **质量把控** - 输出前质量检查
4. **效率优先** - 避免过度分析
