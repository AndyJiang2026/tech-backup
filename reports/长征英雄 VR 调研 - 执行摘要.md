## 📊 交付物三：3-5 种可选合作模式及优劣势

### 模式 1：阶梯分账联营（⭐推荐）
```
授权费：免或 3-5 万/年
分账：月票房≤5 万 (20%), 5-10 万 (25%), ≥10 万 (30%)
保底：无
```
**优势**：低门槛、激励强、风险共担  
**劣势**：前期收益低、需监管系统  
**适用**：二三线城市、文旅/出版集团

### 模式 2：设备租赁 + 内容授权
```
设备：3-5 万/月
内容：5-8 万/年
分账：无或 5 元/票
押金：15-25 万
```
**优势**：现金流稳定、轻资产试水  
**劣势**：长期成本高、缺乏绑定  
**适用**：短期活动、展会、试点

### 模式 3：托管运营
```
授权费：免
分账：甲方 40-50%
保底：3-8 万/月
运营：甲方托管
```
**优势**：质量可控、收益上限高  
**劣势**：运营成本高、需派驻团队  
**适用**：大型文旅项目、政府合作

### 模式 4：项目定制 + 分成
```
定制费：30-80 万
分账：10-15%流水
授权：5-10 年
```
**优势**：一次性收益高、长期稳定  
**劣势**：前期投入大、周期长  
**适用**：政府党建基地、国企展厅

### 模式 5：区域独家代理（优化版）
```
授权费：10-20 万/年（降 50%）
分账：15 元/票（取低值）
任务：年度票房目标
```
**优势**：保持框架、降低门槛  
**劣势**：仍有门槛、需考核  
**适用**：省级文旅/出版集团

## 📊 交付物四：推荐方案及理由

### 🏆 推荐：组合式合作模式（3 档可选）

#### A 档：经济型（福建等经费有限地区）
```
授权费：3 万/年
分账：阶梯 20%-30%
保底：无
合作期：2 年
团体票价：80-120 元/人
```

#### B 档：标准型（主流市场）
```
授权费：10-15 万/年（降 50%）
分账：15 元/票
保底：无或 3 万/年
合作期：3 年
```

#### C 档：定制型（政府/国企大客户）
```
定制费：50-80 万
分账：10%或无
授权：5-10 年
```

### 推荐理由

| 维度 | 说明 |
|-----|------|
| **解决痛点** | 授权费降低 50-85%，无保底压力 |
| **匹配经费** | 票价 80-120 元符合党建/工会标准 |
| **发挥优势** | "龙标"稀缺性 + 革命题材刚需 |
| **市场策略** | 经济型占二三线，标准型固核心，定制型树标杆 |

## 📋 成功案例要点

| 案例 | 模式 | 关键突破 | 成效 |
|-----|------|---------|------|
| 《南昌起义》VR | 政府定制 + 联营 | 纳入党建必选 | 年 15 万人次，900 万 + |
| 《长安十二时辰》 | 联营分成 | IP 流量 + 低门槛 | 全国 12 城，单店 200 万/月 |
| 《只有河南》VR | 托管运营 | 文旅配套 + 专业运营 | 年 80 万人次 |
| 当红齐天迪士尼店 | 城市合伙人 | 核心选址 + 品牌 | 单店 300 万/月，6 个月回本 |

### 成功共性
1. **政府/渠道背书**：纳入党建必选、采购目录
2. **合理定价**：符合经费标准，团体 80-120 元
3. **灵活合作**：低门槛 + 分账激励
4. **打包销售**：与景区/旅行社/研学课程绑定
