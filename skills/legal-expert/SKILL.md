## 📋 合同审查清单

### 必查 7 类条款

| 条款类型 | 审查要点 | 常见风险 |
|---------|---------|---------|
| **权利义务** | 是否对等、明确 | 单方义务过重、权利不对等 |
| **付款条款** | 金额、时间、条件 | 付款周期过短、条件模糊 |
| **知识产权** | 归属、使用范围 | 归属不清、授权过宽 |
| **保密条款** | 期限、范围、违约责任 | 期限过短、无违约责任 |
| **违约责任** | 违约金比例、计算方式 | 违约金过高/过低、不对等 |
| **争议解决** | 管辖法院/仲裁机构 | 管辖地不利、成本过高 |
| **合同解除** | 解除条件、程序 | 解除条件苛刻、无退出机制 |

## 🔴 高风险条款识别

### 霸王条款特征
1. **排除对方权利**：「甲方有权...乙方不得...」
2. **单方解释权**：「本公司保留最终解释权」
3. **不公平违约金**：日息超过万分之五
4. **排他性限制**：「唯一合作方」「不得与其他方合作」
5. **自动续约**：「到期自动续约」无提醒
6. **管辖不利**：约定对方所在地法院

### 修改建议模板
```
原条款：[引用原文]
风险等级：高/中/低
法律依据：《民法典》第 XXX 条
修改建议：[具体修改文本]
谈判策略：[必须修改/建议争取/可协商]
```

## ⚖️ 常用法律条款

### 《民法典》合同编
- 第 465 条：合同约束力
- 第 470 条：合同内容
- 第 509 条：全面履行
- 第 577 条：违约责任
- 第 584 条：损失赔偿
- 第 585 条：违约金调整

### 违约金标准
- 正常范围：日万分之三至万分之五
- 过高标准：超过造成损失的 30%
- 法律依据：《民法典》第 585 条
