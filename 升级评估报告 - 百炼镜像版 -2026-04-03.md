## 🎯 升级方案（安全优先）

### 方案 A：保守安全升级 ⭐⭐⭐⭐⭐（强烈推荐）

**适用场景**: 生产环境，数据安全第一

#### 步骤 1: 完整备份（必须）

```bash
# 1.1 备份配置文件
sudo cp /home/admin/.openclaw/openclaw.json \
  /home/admin/.openclaw/openclaw.json.backup.20260403.$(date +%H%M)

# 1.2 备份整个 .openclaw 目录（可选但推荐）
tar -czf /tmp/openclaw-backup-$(date +%Y%m%d).tar.gz \
  --exclude='extensions' \
  -C /home/admin/.openclaw .

# 1.3 备份工作空间
tar -czf /tmp/workspace-backup-$(date +%Y%m%d).tar.gz \
  -C /home/admin/.openclaw workspace-chief-agent

# 1.4 验证备份
ls -lh /tmp/openclaw-backup-*.tar.gz
ls -lh /tmp/workspace-backup-*.tar.gz
```

**预计时间**: 2-3 分钟  
**备份大小**: 约 3MB（不含 extensions）
