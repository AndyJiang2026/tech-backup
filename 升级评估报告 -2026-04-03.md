## 📝 升级步骤建议

### 方案 A：保守升级（推荐）⭐

**适用场景**: 生产环境，稳定性优先

```bash
# 1. 备份配置（必须）
cp /home/admin/.openclaw/openclaw.json /home/admin/.openclaw/openclaw.json.backup.$(date +%Y%m%d)

# 2. 升级 npm（推荐）
sudo npm install -g npm@latest

# 3. 升级 OpenClaw
sudo npm install -g openclaw@latest

# 4. 验证版本
openclaw --version

# 5. 检查配置
openclaw config check

# 6. 重启 Gateway
openclaw gateway restart

# 7. 验证功能
openclaw gateway status
```

**预计时间**: 10-15 分钟  
**风险等级**: 🟡 中
