### 2️⃣ PortEden Email 配置

**用途**: 邮件管理功能

**方案 A - 本地电脑登录后复制 Token**（推荐）:
```
1. 在你的本地电脑（Windows/Mac，有浏览器）上安装 PortEden CLI:
   # macOS
   brew install porteden
   
   # Windows (PowerShell)
   winget install PortEden.CLI

2. 登录获取 token:
   porteden auth login

3. 查看 token:
   porteden auth status
   # 会显示你的 API token

4. 复制 token，然后在服务器上执行:
   porteden auth login --token pe_xxxxx
```

**方案 B - 网页获取 Token**:
```
1. 访问：https://my.porteden.com/settings/api-keys
2. 创建新的 API Key
3. 复制 token，在服务器执行:
   porteden auth login --token pe_xxxxx
```

**验证**:
```bash
porteden auth status
porteden email messages --today -jc
```
