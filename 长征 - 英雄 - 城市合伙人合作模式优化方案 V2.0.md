# 《长征·英雄》城市合伙人合作模式优化方案 V2.0

**编制日期**：2026 年 4 月 13 日  
**优化背景**：福建/咸阳/内蒙古三地测算反馈"难以回本"  
**核心目标**：降低门槛 + 多元收入 + 快速回本

---

## 一、原合作模式问题分析

### 1.1 原模式（已验证不可行）

```
❌ 50 万/地市 IP 授权费 + 30 元门票分账

回本测算（以福建海峡出版为例）：
- 授权费：50 万
- 运营成本：20 万/年（场地 + 人力 + 运维）
- 门票分账：30 元/人次
- 年客流：8000 人次（预估）
- 年收入：24 万/年
- 回本周期：3.5 年 ❌（客户不可接受）
```

### 1.2 客户反馈

| 客户 | 反馈 |
|------|------|
| 福建海峡出版 | "除非省里有政策文件，否则难以回本" |
| 咸阳文旅集团 | "红色研学季节性明显，淡季亏损" |
| 内蒙古广电 | "地域广阔客流分散，50 万授权费过高" |

### 1.3 核心问题

| 问题 | 权重 | 说明 |
|------|------|------|
| 授权费过高 | 40% | 50 万 upfront，客户心理门槛高 |
| 收入来源单一 | 30% | 仅靠门票分账，抗风险能力弱 |
| 客群定位模糊 | 20% | 党员/红培/国防教育未差异化运营 |
| 增值服务缺失 | 10% | 无培训/认证/申报等增值收入 |

## 二、优化后合作模式（V2.0）

### 2.1 三层合作模式设计

```
┌─────────────────────────────────────────────────────┐
│  轻量版        标准版         旗舰版              │
│  15 万/年      30 万/年       80 万/3 年           │
│  低门槛试用    主流选择       深度绑定            │
└─────────────────────────────────────────────────────┘
```

### 2.2 轻量版（15 万/年）

| 项目 | 内容 |
|------|------|
| **授权费** | 15 万元/年（降低 70% 门槛） |
| **授权范围** | 单城市单场馆 |
| **门票分账** | 30 元/人次 |
| **基础服务** | 内容更新 + 平台维护 + 远程运维 |
| **适合客户** | 试探性合作、客流不确定客户 |
| **回本周期** | 5000 人次/年（约 1.5 年回本） |

**优势**：低门槛试水，降低决策风险

### 2.3 标准版（30 万/年）⭐推荐

| 项目 | 内容 |
|------|------|
| **授权费** | 30 万元/年 |
| **授权范围** | 单城市单场馆 |
| **门票分账** | 25 元/人次（优惠 17%） |
| **增值服务** | 党员培训认证 + 红培研学课程包 |
| **政策申报** | 协助申报省级思政/研学基地（20-50 万补贴） |
| **适合客户** | 主流客户（福建/咸阳/内蒙古级别） |
| **回本周期** | 8000 人次/年 + 增值服务（约 1 年回本） |

**优势**：平衡投入产出，增值服务提升收益

### 2.4 旗舰版（80 万/3 年）

| 项目 | 内容 |
|------|------|
| **授权费** | 80 万元/3 年（相当于 26.7 万/年） |
| **授权范围** | 单城市独家 + 优先续约权 |
| **门票分账** | 20 元/人次（优惠 33%） |
| **增值服务** | 全部增值服务（培训 + 认证 + 申报 + 定制） |
| **定制内容** | 本地红色资源 VR 化（1 个项目） |
| **适合客户** | 省级出版集团/大型文旅集团 |
| **回本周期** | 1 万 人次/年 + 增值服务（约 8 个月回本） |

**优势**：深度绑定，成本最优，独家保护

## 三、多元收入结构设计

### 3.1 收入来源对比

| 收入类型 | 原模式 | V2.0 优化 |
|----------|--------|----------|
| 门票分账 | ✅ 30 元/人次 | ✅ 20-30 元/人次（分层） |
| 党员培训 | ❌ 无 | ✅ 500 元/人次（认证费） |
| 红培研学 | ❌ 无 | ✅ 200 元/学生（课程包） |
| 国防教育 | ❌ 无 | ✅ 300 元/学生（军训拓展） |
| 政策申报 | ❌ 无 | ✅ 20-50 万/项目（补贴） |
| 定制内容 | ❌ 无 | ✅ 10-30 万/项目 |

### 3.2 标准版收入测算（以福建海峡出版为例）

| 收入来源 | 单价 | 年人次/项目 | 年收入 |
|----------|------|------------|--------|
| 门票分账 | 25 元 | 8000 人次 | 20 万 |
| 党员培训 | 500 元 | 200 人次 | 10 万 |
| 红培研学 | 200 元 | 1000 学生 | 20 万 |
| 国防教育 | 300 元 | 300 学生 | 9 万 |
| 政策申报 | 30 万 | 1 项目/年 | 30 万 |
| **合计** | - | - | **89 万** |

**成本**：30 万授权费 + 20 万运营 = 50 万/年  
**毛利**：89 万 - 50 万 = 39 万/年  
**回本周期**：第一年即盈利 ✅

## 七、销售话术优化

### 7.1 原话术（已验证无效）

```
❌ "50 万授权费，30 元门票分账，客流做起来就回本了"

问题：客户测算后认为难以回本
```

### 7.2 新话术（V2.0）

```
✅ "15 万起合作，多元收入结构，1 年回本"

话术要点：
1. 低门槛：15 万/年起，降低决策风险
2. 多元收入：门票 + 培训 + 研学 + 申报，不依赖单一客流
3. 政策补贴：协助申报 20-50 万补贴，实际成本更低
4. 回本保障：标准版 1 年回本，旗舰版 8 个月回本
```

### 7.3 分客群话术

**党员培训客户**：
> "500 元/人次认证费，200 人次就是 10 万收入，加上门票分账，1 年回本没问题"

**红培研学客户**：
> "200 元/学生，寒暑假各 500 学生就是 20 万，还不算门票和党员培训"

**文旅集团客户**：
> "协助申报文旅融合示范项目，成功就是 30-100 万补贴，授权费直接覆盖"
